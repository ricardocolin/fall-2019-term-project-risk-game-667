## Development

1. Create .env file that looks the following way

```
DATABASE_URL=postgres://<db_user>:<db_password>@<db_host>:<db_port>/<db_name>
```

2. In first terminal window, start Webpack in watch mode

```
npm run build:watch 
```

2. In second window, start Express 

On Linux

```
npm run start:dev 
```

On Windows

```
npm run start:win 
```
